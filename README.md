# Sistema de Inventarios — Rocoto · Hot Wings · Malanga

Página web de inventario para 3 marcas con listado de insumos editable por categoría.

## Características
- Pestañas independientes por marca (Rocoto, Hot Wings, Malanga)
- Búsqueda en tiempo real por nombre o código
- Filtro por categoría
- Campo de cantidad editable por insumo
- Estadísticas en vivo (total, mostrados, con cantidad)

## Uso
Abrir `index.html` directamente en el navegador. No requiere servidor ni dependencias externas.

## Despliegue en GitHub Pages
1. Sube el repositorio a GitHub
2. Ve a **Settings → Pages**
3. Selecciona la rama `main` y carpeta `/root`
4. El sitio quedará disponible en `https://tu-usuario.github.io/nombre-repo`
